### WiFi 접속설정 파일 만들기

-   boot 파티션에 wpa_supplicant.conf 작성

```
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
network={
    ssid="네트워크이름"
    psk="네트워크비번"
}
```

부팅 후 우측 상단 와이파이 아이콘에 마우스 올리면 IP 주소 확인 가능
